# talking_heads/

Non-manim video segments for the yahtzee video — the talking-head clips (THA–THL)
that get cut into the final edit alongside the manim renders.

The clips themselves (`.mp4` / `.mov` / …) are **gitignored** (large binaries — see
the repo `.gitignore`), so git tracks only this folder + this README, not the
videos. Drop segment clips here locally; they are not committed or synced by git.
