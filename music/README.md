# music/

Background music / audio tracks for the yahtzee video — the source tracks laid
under the final edit alongside the manim renders and voiceover.

The tracks themselves (`.mp3` / `.wav` / …) are **gitignored** (large binaries — see
the repo `.gitignore`), so git tracks only this folder + this README, not the
audio. Drop tracks here locally; they are not committed or synced by git.
