# footage/

Non-manim video for the yahtzee video — the source clips (for yahtzee, the
talking-head segments THA–THL) cut into the final edit alongside the manim renders.

The clips themselves (`.mp4` / `.mov` / …) are **gitignored** (large binaries — see
the repo `.gitignore`), so git tracks only this folder + this README, not the
videos. Drop clips here locally; they are not committed or synced by git.
